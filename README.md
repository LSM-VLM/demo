# LSM-VLM Demo Page

This repository contains a static project page for LSM-VLM.

The page is a Bulma-based academic project page with a centered hero, figure
cards, concise method sections, result figures, and placeholder blocks for
future public paper/code links.

## Assets

- `static/images/teaser.png`: LSM-VLM teaser.
- `static/images/pipeline.png`: LSM-VLM pipeline overview.
- `static/images/bev-example.png`: local BEV occupancy example.
- `static/images/sample-view.jpg`: sample egocentric observation.
- `static/images/main-results-radar-gap.png`: best-gap radar chart.
- `static/images/main-results-gap-heatmap.png`: per-task gap heatmap.
- `static/images/main-results-dumbbell.png`: main result dumbbell plot.
- `static/images/ablation-vsi-radar.png`: component ablation radar chart.

## Open Locally

Open `index.html` directly in a browser, or serve this directory with any static
file server.
